# CS106L-Learning-2025-fall
记录了本人自学完cs106L的全部过程，上传了lecture和笔记，使用的版本是2025fall，贴一个网址：https://web.stanford.edu/class/cs106l/

这是我真正意义上自己独立自主学习完的一个公开课。

cs106L是讲现代c++的，在standford是与cs106B配套一起学的，我从这个课程里，学到匿名函数是怎么设计的，以及c++特有的流，还有现代c++对于类的设计和模版的写法，当然，还有一些智能指针，RAII，optional等等，以及很多原则和c++的哲学，如rules of zero/three/five

其实学校讲的面向对象编程还是太简单了，这个概念以及这些年发展成一个非常庞大的内容，我们需要学习前沿之后才能更好理解作者 那种希望c++强大的同时也兼具安全的思想；以及那种很强的实用主义和优化。

经过这次学习，我感觉我学到很多关于自学的实践，未来学习更多自己想学的东西也知道怎么开展了，比如可以先阅读材料，有条件可以看看视频，但是关键是自己理解，整理笔记，以及自己讲授内容，都是相当好的方法，未来可能有时间还会继续完善这个讲课，自己录制到lecture11可能没时间了，未来可能继续学习cs61c等等内容，期待未来会更好。
